## ğŸ“Š Dataset Overview

The **Iris** dataset contains 150 observations of iris flowers from three species: *Setosa*, *Versicolor*, and *Virginica*. It includes the following features:

- **sepal_length (cm)**
- **sepal_width (cm)**
- **petal_length (cm)**
- **petal_width (cm)**
- **species** (categorical)
